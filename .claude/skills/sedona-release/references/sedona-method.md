# Sedona Method — Core Reference

## The Four Wants (四种核心欲望)

These are the root-level wants that drive all suffering and attachment. Every negative emotion, when traced far enough, collapses into one or more of these four.

### 1. 控制欲 (Wanting Control / Wanting to Change)
- **Core feeling**: "This shouldn't be happening. I need to fix this. I need things to be different."
- **Trigger phrases**: "我想改变这个情况" / "这不公平" / "我必须解决这个" / "事情应该是___"
- **Body signs**: Chest tightness, jaw clenching, restless hands, urgency
- **Releasing question**: "你是否想要控制或改变这个情况？"

### 2. 认同欲 (Wanting Approval / Wanting to Be Seen)
- **Core feeling**: "I need them to understand me. I need their validation. I need to be right."
- **Trigger phrases**: "他们不理解我" / "我需要别人认可" / "我不想被评判" / "我想证明自己"
- **Body signs**: Throat tightness, heart heaviness, face flushing, shrinking posture
- **Releasing question**: "你是否想要别人的认可或理解？"

### 3. 安全感欲 (Wanting Security / Wanting Certainty)
- **Core feeling**: "I don't know what will happen. I need guarantees. I'm afraid of losing what I have."
- **Trigger phrases**: "我不确定未来" / "如果___怎么办" / "我怕失去___" / "我需要知道结果"
- **Body signs**: Stomach knots, shallow breathing, scanning/vigilance, freeze
- **Releasing question**: "你是否想要更多确定感、保障或安全感？"

### 4. 存在感欲 (Wanting to Be Separate / Wanting to Exist as This Identity)
- **Core feeling**: "This is who I am. If I let this go, I won't be me. I need to protect this identity."
- **Trigger phrases**: "这就是我的性格" / "我一直都是这样" / "这是我的问题" / "我不知道没有这个的自己是谁"
- **Body signs**: Rigidity, resistance to releasing itself, "but what if" loops
- **Releasing question**: "你是否觉得这个情绪/故事是你的一部分，或者定义了你是谁？"

## The Three Releasing Questions (三问法)

These three questions are the core of the Sedona Method. Ask them in sequence, slowly, with genuine curiosity — not as a performance.

```
问题一: "你能放下这种欲望吗？"
        (Could you let this wanting go? / Is it possible to let this go?)
        → Just checking if it's possible. "Yes" or "No" both fine.

问题二: "你愿意放下它吗？"
        (Would you let it go? / Are you willing?)
        → Checking willingness. If "No" — that's okay, welcome the resistance.

问题三: "你会在什么时候放下它？"
        (When would you let it go? / Starting when?)
        → Open the door to "now."
        → If resistance: "好的，那我们先继续欢迎这个抗拒。"
```

**Important**: These questions are not about forcing release. They simply open the door. The release happens by itself when the wanting is seen clearly.

## Recognizing Release (释放的信号)

A genuine release often brings one or more of these signs:
- 突然的轻松感 — a sense of lightness, like something lifted
- 深呼吸 — spontaneous deep breath or sigh
- 笑意或泪水 — unexpected laughter or tears (both are release)
- 沉默 — a moment of quiet where the thought simply doesn't arise
- 身体松弛 — shoulders drop, jaw unclenches, stomach softens
- 问题感觉不重要了 — the original problem feels smaller, less charged

**If no release**: Don't push. The resistance itself is the next thing to welcome. Ask: "你能欢迎这种'不想放下'的感觉吗？"

## The Onion Principle (洋葱原则 — for Mode 2)

Each release peels one layer. The surface emotion protects a deeper want. The pattern:

```
表层情绪 (Surface emotion)
    ↓ 释放
次层欲望 (Secondary want)
    ↓ 释放
核心恐惧 (Core fear)
    ↓ 释放
最深的欲望 (Root want — one of the four)
    ↓ 释放
自由 (Freedom)
```

When doing卡点深挖, after each release ask: "在这之下，还有什么感受或欲望在等待被看见？"

## Language Principles (引导语言原则)

1. **Never invalidate** — Never say "你不应该感受到这个" or suggest the feeling is wrong
2. **Welcome everything** — Every feeling, including resistance, is welcomed, not fought
3. **Pace matters** — Speak slowly, create space between questions, let silence exist
4. **Check-in often** — "此刻感觉如何？" after every major release attempt
5. **Freedom score** — Use 0-10 scale: 0 = 完全束缚 (completely stuck), 10 = 完全自由 (completely free)
6. **No "should"** — Replace "你应该感受到..." with "有些人在释放后会..."
